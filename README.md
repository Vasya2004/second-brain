# Second Brain — Василий

Единая база знаний. Читается целиком AI (Claude) для получения полного контекста.

---

## Структура

```
/second-brain
  README.md              ← этот файл, точка входа
  /Заметки               ← исходный архив (не изменять)
  /about-me              ← кто я, цели, ценности, контекст для AI
  /projects              ← мои текущие и завершённые проекты
  /knowledge             ← база знаний по темам
    /psychology          — мышление, дисциплина, самопознание
    /health              — тело, питание, иммунитет, долголетие
    /business            — бизнес-модели, метрики, монетизация
    /ai-future           — AI, рынок труда, власть, риски
    /content-blog        — блог, контент, монетизация аудитории
```

---

## Навигация для AI

**Чтобы понять, кто я:** читай `/about-me/README.md`

**Чтобы понять мои проекты:** читай `/projects/`

**Чтобы получить базу знаний по теме:**
- Психология и продуктивность → `/knowledge/psychology/`
- Здоровье и тело → `/knowledge/health/`
- Бизнес → `/knowledge/business/`
- AI и будущее → `/knowledge/ai-future/`
- Контент и блог → `/knowledge/content-blog/`

**Исходные заметки (первоисточники):** `/Заметки/` — не изменяются, служат архивом.

---

## Принципы ведения базы

- Каждый файл в `/knowledge/` — самодостаточный документ по подтеме
- Обычный markdown, без wiki-ссылок, читается целиком
- Папки в `/knowledge/` расширяемые — новые заметки из курсов докидывай туда
- `/Заметки/` не трогать — исходный архив остаётся неизменным
