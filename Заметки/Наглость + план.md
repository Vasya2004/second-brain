## Амбиции без стратегии — это эго. Стратегия без амбиций — это паралич

Работает только связка: наглость + план. Амбициозный человек без стратегии бьётся в эго и теряется. Стратег без наглости сидит с идеальным планом но никогда не действует. Двигаются вперёд только те у кого есть оба компонента одновременно.
