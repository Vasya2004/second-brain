## Суть продукта
Lifera — личная Life RPG система. Не таск-менеджер и не трекер привычек — полноценная игровая механика поверх жизни.

Основной цикл:
**Цель → Миссия → Прогресс → Опыт (XP) → Уровень → Достижения → Рекомендация ассистента**

Разделы продукта: Цели, Миссии, Навыки, Здоровье, Финансы, Достижения, Ассистент Lifera, Профиль, Настройки.

Подписки: Free / Pro / Ultra.

## Стек
- Next.js 16 App Router
- React 19
- TypeScript
- Tailwind CSS 4
- Supabase (Auth + PostgreSQL + RLS)
- Vercel (хостинг)
- OpenAI API (зарезервирован для будущего LLM-ассистента; текущий ассистент — rule-based)
